Analyse de l'exercice

Connaissances

Son principal intérêt réside dans la maîtrise de l'événement scroll en JavaScript. Savoir intercepter l'état de défilement d'un élément (scrollTop) pour le traduire en une action visuelle (changement de couleur) est une compétence très utile.

Critique

C'est un exercice court qui prouve la capacité à lier l'interaction utilisateur à un style dynamique, ce qui peut ouvrir la voit à des possibilités plus poussé en terme de visuel pour la création de page.

Difficulté : 2/5.

Exercice 6 : Effet Parallaxe

Effet de Profondeur

L'image de fond semble rester fixée tandis que ce contenu défile par-dessus.

Bloc de séparation neutre pour accentuer l'effet.

CSS Pur

Réalisé uniquement avec background-attachment: fixed.

Analyse de l'exercice

Connaissances

J'ai utilisé la propriété CSS background-attachment: fixed. Elle permet de fixer l'image d'arrière-plan par rapport à la fenêtre du navigateur, créant ainsi un décalage de mouvement avec le contenu qui défile.

Critique

Cet exercice est de loin celui qui offre le meilleur rendu visuel. L'effet Parallaxe améliore l'immersion et le design général d'une page. Le code CSS est très simple mais marche très bien et donne le résultat voulu.

Difficulté : 2/5.
