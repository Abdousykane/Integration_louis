Travaux Pratiques N°6

Objectif : Animations CSS et manipulations dynamiques via JavaScript.

Exercice 1 : Effet Rideau (Transition)

Survolez l'image pour ouvrir les rideaux.

Analyse de l'exercice

Connaissances

J'ai appris à utiliser la propriété transition: transform pour animer le déplacement des éléments. L'usage de position: absolute permet de superposer les rideaux sur l'image, et overflow: hidden sur le conteneur assure que les rideaux disparaissent proprement lorsqu'ils sortent du cadre.

Critique

L'effet est simple mais efficace pour révéler du contenu. La fluidité est gérée nativement par le navigateur usage du JavaScript.

Difficulté : 2/5.

Exercice 4 : Déformation Aléatoire (Matrix)

Cliquez sur "Start" pour lancer une déformation aléatoire de l'image via JavaScript.

Analyse de l'exercice

Connaissances

J'ai manipulé le DOM via JavaScript pour appliquer dynamiquement la propriété CSS transform: matrix(...) . L'utilisation de setInterval permet de renouveler l'effet à intervalle régulier, tandis que la transition CSS assure l'interpolation fluide entre les états.

Critique

Cet exercice montre comment JS et CSS marchent très bien une fois les deux utilisé pour gérer l'aspect visuel d'un objet de manière dynamique : JS calcule les valeurs aléatoires, CSS gère le rendu fluide.

Difficulté : 3/5.

Exercice 5 : Bannière animée (728x90px)

Ready to Fight!

Analyse de l'exercice

Connaissances

J'ai combiné plusieurs techniques d'animation : @keyframes pour créer des mouvements continus (pulsation, rotation) et des entrées d'éléments (slide). Le tout est orchestré avec animation-delay pour séquencer l'apparition des éléments.

Critique

C'était un exercice intéressant car il demandanit d'être créatif et de savoir comment donner vie à notre vision, pour ce qui est du code j'apprécie vraiment ce que j'ai pu faire.

Difficulté : 5/5.

Ce qui était vraiment dur n’était pas de trouver l'inspiration pour le concept, mais de l'exécuter ! Il faut savoir retrouver les propriétés CSS adéquates et comprendre comment les utiliser. Par exemple, il faut savoir régler parfaitement les questions de temps (animation-duration), de positionnement (left ou transform), et de rotations (rotate) pour que tous les éléments s'harmonisent correctement. Ce n’était vraiment pas facile à créer cet équilibre, mais j’ai pu avoir un résultat satisfaisant.
