Analyse de l'exercice

Connaissances

Cet exercice m'a permis de réviser la structure des tableaux HTML complexes (utilisation de rowspan et colspan pour fusionner cellules et colonnes). J'ai également mis en pratique les propriétés Flexbox ( justify-content , align-items ) non seulement pour reproduire le mémo visuel, mais aussi pour structurer l'en-tête de l'exercice.

Critique

Je pense que cet exercice est une bonne porte d'entrée quant à l'utilisation de la Flexbox qui est une notion indispensable pour manipuler l'emplacement de divers élèment du CSS.

Du côté fonctionnel, j'ai réussi à afficher un résultat identique à celui demandé.



Difficulté : 2/5.
j'ai eu quelques difficultés avec le tableau d'en bas (notamment la cellule du "flex: 0 1 auto") mais la petite demonstartion faite en TD m'a permis de mieux y voir clair afin de réussir cette exercice.

Analyse de l'exercice

Connaissances

J'ai appris à combiner différentes techniques de positionnement : position: absolute pour placer le menu dans le header, background pour mettre les images en format GIF et display: flex pour créer la mise en page principale avec une barre latérale (aside) et un contenu principal.

Critique

Je trouve cette exercice pertinent car la page a un aspect similaire a plusieurs type de documents qu'on peut trouver sur internet comme les CVs, faire le css de cette page m'a permis donc de mieux comprendre comment ces pages fonctionnait et je pourrais peut-être m'inspirer pour de futurs travaux.

Le rendu est fonctionnel et réactif grâce au Flexbox sur le conteneur principal.

Difficulté : 3/5.
Cette exercice ne fut pas évident au premier abord mais quelques petites réalisations comme l'usage de la propriété background permettent de s'en sortir et de ce rendre compte le code CSS est relativement léger.
